| 属性名 |   含义      |   类型     |  是否必填 | 默认值  |

# ImageLoader

<img src = " " alt = " " style = "zoom:50%;">

## 属性

| 属性名 |   含义      |   类型    |  是否必填 | 默认值  | 
|------ | ---------- | -------- | --------- | -------|
| url  |  原始图片地址   |  String  |  是   |    无    | 
| placeholder| 占位图片地址| String |  是   |    无    | 
| duration  | 原始图片加载完成，切换到原始图片需要的时间| String |  否   |    1000    |

| 事件名 |   含义      |   事件参数     |  参数类型 |
|------ | ---------- | -------- | --------- |
| imageLoad |   原始图片加载完成  | 无  |  无    |
