## 属性
| 属性名 |   含义      |   类型     |  是否必填 | 默认值  | 
|------ | ---------- | -------- | --------- | -------|
| url   |  头像的地址  |  String  |  是   |    无    | 
| size   |  头像的尺寸，宽高相同  |  Number  |  否   |    150  |
